# Mobile Now API

该项目是基于koa2框架实现的简单的电商平台接口，可以计算订单运费

## Tech Stack

* Koa2
* middlewares
  * koa-router
  * koa-bodyparser
  * koa2-validation

## Getting Start

```
git clone https://github.com/fermin/mobile-now-api.git
cd mobile-now-api
npm install
npm start
```
在浏览器中打开 http://127.0.0.1:3000/

## Test

```
cd mobile-now-api
npm test
```
